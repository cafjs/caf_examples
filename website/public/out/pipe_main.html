pipe_main

Main processing pipeline for CA messages.

Based on express/connect middleware.

Configuration in framework.json:

"pipe": {
   "module": "pipe_main",
   "name": "pipe",
   "env": {
      "relPublicPath" : "../../../public",
      "allowInsecureGET" : false
   }
 }

where:

  • "relPublicPath" is a relative directory path starting from where this file will be found at run-time (typically <yourapptopdir>/nodemodules/cafcore/lib/pipe_main.js) to the public directory with your app static web pages (e.g., <yourapptopdir>/public).

  • "allowInsecureGET" Whether we can see the internal state of a CA from a browser (dangerous!).
