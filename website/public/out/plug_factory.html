instance(caId, env, allowCreate, cb0)

Creates a CA or finds a reference to an existing local one given the CA's id.

If there is already a remote CA with the same id, it returns an error object in the callback with the remote hosting node or a null object if it is missing but we are not allowed to create one.

Parameters:
Name Type Description
caId string

An identifier for the CA.
env Object

A set of properties to configure the new CA. Note that 'env' is ignored if the CA has already been created.
allowCreate boolean

Whether we can create missing CAs.
cb0 function((Object | {remoteNode:string})=, Object=)

A callback returning an error (first argument) that could describe a remote location for the CA (or null location if we cannot create it) or a (second argument) with the CA.
